üì± GPSLogger Setup

GPSLogger is a free Android app that can automatically log your location and send it to Reitti.

Setup Instructions:

  1. Download GPSLogger from the Google Play Store
  2. Open GPSLogger and go to Logging details ‚Üí Log to custom URL
  3. Enable "Log to custom URL"
  4. Set the URL to: https://your-domain.com/api/v1/location-data
  5. Set HTTP Method to POST
  6. In HTTP Headers, add: Authorization: Bearer YOUR_API_TOKEN
  7. Set HTTP Body to: {"points":[{"latitude":%LAT,"longitude":%LON,"timestamp":"%ISODATE","accuracyMeters":%ACC}]}
  8. Set Content-Type header to: application/json
  9. Configure logging frequency as desired (e.g., every 30 seconds)
  10. Start logging!

üó∫Ô∏è OwnTracks Setup

OwnTracks is a privacy-focused location tracking app available for iOS and Android.

Setup Instructions:

  1. Download OwnTracks from the App Store or Google Play Store
  2. Open OwnTracks and go to Settings ‚Üí Connection
  3. Set Mode to HTTP
  4. Set Host to: https://your-domain.com
  5. Set Path to: /api/v1/owntracks
  6. Enable Authentication
  7. Set Username to your Reitti username
  8. Set Password to: YOUR_API_TOKEN
  9. Configure tracking settings as desired
  10. The app will automatically start sending location updates
